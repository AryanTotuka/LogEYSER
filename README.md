# LogEYSER - Real-Time IDS Dashboard

A real-time network intrusion detection system that captures, analyzes, and visualizes network traffic.

## Features
- Live packet monitoring using TShark
- CSV log analysis
- Detection of DoS, brute force, and port scanning attacks
- Interactive dashboard with Chart.js

## Tech Stack
Node.js, Express.js, Socket.IO, Chart.js, TShark, Multer, csv-parser

## Run Project
npm install
node server.js
